오시는길

ReSEAT 프로그램을 운영하고 있는 한국과학기술정보연구원의 약도입니다.

  • 서울
  • 대전

주소 : (130-741) 서울특별시 동대문구 회기로 66

대표전화 : 02-3299-6231,6409

자가운전자

동대문에서 청량리 방면으로 오다가 청량리역 교차로에서 좌회전, 국립산립과학원앞 삼거리에서 세종대왕 기념관을 따라 우회전(경희대방향)하여 15km진입하면 우측으로 한국과학기술정보연구원 건물이 보임


고대에서 미아리 방향으로 가다가 국립산림과학원 방향으로 우회전, 국립산림과학원을 지나 약 15m 앞에 우측으로 한국과학기술정보연구원 건물이 보임

지하철

1호선 청량리역 2번 출구(청량리 우체국 방면)

6호선 고려대(종암)역 3번 출구

버스

파랑(간선)버스 273

초록(지선)버스 1215, 1226

주소 : (305-806) 대전광역시 유성구 대학로 245

대표전화 : 042-869-0685

자가운전자

대덕밸리 톨게이트 이용시

경부고속도로 → 화덕인터체인지 → 호남고속도로 → '대덕밸리 톨게이트' 나와 좌회전 → 곧이어 굴다리 아래로 지나며, 사거리에서 직진(연구단지 방면) → 대덕 롯데 호텔에서 우회전 → 직진 → 엑스포과학공원(대덕대교)에서 우회전 → 국립과학관 지나 3Km지점 → KAIST 정문진입 → 좌회전 → 250m 지나면 우측으로 원형건물(KISTI)이 보임


유성 톨게이트 이용시

'유성톨게이트'나와 우회전 사거리 우회전 계속 직진 지하차도로 들어가지 말고 우측진행 충남대학교 정문(좌측방면)에서 직진(좌측방면의 도로임) → 유성구청 KAIST 정문진입 좌회전 250m 지나면 우측으로 원형건물(KISTI)이 보임

지하철

1호선 월평역 3번출구(도보 15분소요) → KAIST 하차

버스

지선버스 104(시청역), 121(유성역) → KAIST 하차
